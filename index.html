Main Project Size

가장 대중적인 사이즈로 작업합니다.

모바일

-모바일의 양 마진은 컨텐츠의 집중을 위해 가장 최소한의 마진값을 잡았습니다.
-텍스트 min 사이즈는 16px / 라디오 옵션 텍스트 min 14px로 읽는 것에 불편함이 없게합니다.

웹

-최소 300px 이상의 여백을 두고 있습니다. Box의 각 양 옆의 마진값은 15 ~ 20px 이하로 지정합니다. 아이템을 붙이고 각 섹션의 컨텐츠의 마진을 높게 주는 것이 더 그룹핑하기 쉽다고 생각합니다.

Button Size

가장 안정적인 사이즈로 작업합니다.

Margin Size 14px 이상

텍스트와 버튼 bg가 붙지 않고 여유있게하기 위해 10p + 4px 이상을 띄웁니다.

Improtant Font Size

가장 안락한 사이즈로 모십니다.

Noto Sans KR = 한글

Font Size: 평균 16px / 최소 14px
